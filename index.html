The hardest thing that I have learned so far this semester but I now understand is CSS layouts. The layout is how each element appears on a page. These layouts can be manipulated through simple programming to make the page more visually appealing. Some basic commands in constructing layouts are boxes, display type, position, and floats. These elements make up the majority of CSS layouts and can easily be manipulated to change the appearance of a webpage. Boxes display type and content and include margin, border, and padding. Display type allows you to adjust the height and width of a box according to the length of the content in the box. Position determines where the text appears on the page and it has multiple types: absolute (element moves as you scroll), fixed (element does not move), and sticky (which is an elastic form of fixed position). Floats shift elements left or right on a page and other elements will follow around the float. Some types of floats include float left (elements shift from right to left) and float right (elements float from left to right). These are just some of the basic elements of CSS layouts, but I feel like this is the most complex thing I have learned so far that I feel confident in my ability to use it.
